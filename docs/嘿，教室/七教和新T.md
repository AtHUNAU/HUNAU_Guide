# 七教和新T
目前最老的教学楼，但每间教室都有效果带劲的中央空调。

## 位置
七教在芷兰园区的南面，出了芷兰南门就是。具体可以高德搜索“第七教学楼”。七教分南北楼，注意区分。
### 新T2在哪？
如果教室为“新T”，那这些教室在第七教学楼的南侧。第一次找可以先进入七教南楼1楼，一直往里面走，然后跟随指引牌，就能找到了。

## 厕所
七教北的厕所位于走廊两端尽头。七教南的厕所位于走廊的西侧尽头。

## 楼梯
七教南的楼梯位于走廊的东侧尽头和走廊中间，七教北的楼梯位于走廊两端尽头和走廊中间。

## 电梯
电梯位于七教南。

## 外卖架
外卖架位于七教的广场上，出了门直走左手边树下，在室外。
