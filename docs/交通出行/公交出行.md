# 公交出行

## 在学校里坐车
学校里有“宝宝巴士”，绕学校一圈，串联起宿舍区、教学楼和大门。要付费乘车。

## 坐地铁出门
学校边上有长沙地铁6号线“农科院农大站”，能方便的前往长沙老城区。
但地铁站离学校大门有一段距离，学校提供地铁接驳小巴，也可以自行骑车前往地铁站。

## 坐公交出门
学校北门外有“湖南农大公交首末站”，有较多公交线路首末站设在此处。

西门外有公交站“湖南农业大学西门”，南门外有公交站“东沙古井”和“轩逸小区”。

## 去火车站
去长沙火车站可以坐地铁6号线转2号线，或者北门坐10路公交车。

去长沙南站，可以在西门外乘坐280路、X117路、X111路公交车。但更推荐网约车去南站，一般20不到。

## 滴，交通卡
非常推荐在手机的“钱包”app里开通长沙的电子公交卡，公交7折地铁9折，充值后只需要把手机对上卡机/闸机就可以，别的都不用做。

也可以使用微信/支付宝的乘车码，但每次都要手动打开。
