# 校外饮食和购物

## 红旗市场
红旗市场在学校的西北角，可以直接导航过去，里面有很多店铺。有一家乐尔乐超市在红旗8栋2楼，价格相当的实惠。

## 东湖小区
北门出去后马路对面的小区，也有很多店铺和摆摊的，可以自行探索。

## 夜经济
夜经济是北门出门后一直往前走的左手边，楼下是[快递驿站](/日常生活/快递)和乐尔乐超市（依然价格实惠），楼上则有很多吃的。

## 星都汇
出北门一直往前走的右手边，有肯德基的那片。有很多小饭馆子，也有健身馆ktv网吧一类的。

## 生机创业街
在农大公交首末站的位置，出北门右转后一直走，看到一家零食店就是。也有很多店子（但数量比上面几个少）。

## 龙湖芙蓉天街
离学校稍远，跟着导航走。很标准的现代商场，吃喝玩乐一站俱全。

## 更远的地方
- 出西门的公交站，可以坐上X111路公交车到星沙中学下车，就是星沙万象汇。不堵车的话从上车到下车只要15分钟；
- 出西门马路对面的公交站，可以坐上273路到“劳动路花候路口”去高铁吾悦广场；
- 周边其他商圈像运达汇、万家丽则自行大胆探索。
