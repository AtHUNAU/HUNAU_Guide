# 转专业

## 什么时候转？
根据最新的[转专业管理办法](https://jwc.hunau.edu.cn/gzzd/xxj/202605/t20260525_519816.html)，每年12月份，大一和大二的学生可以申请转专业。

## 谁能转？谁不能转？
除了“以特殊招生形式录取的学生；以艺术、体育类、职高对口、中外合作办学等特殊类别录取入校，申请转入非对应类别专业的；正在休学、保留学籍、保留入学资格或按学校规定应作退学处理的；从外校转入、专升本录取的学生；在校期间已完成过1 次转专业的；其他有违公平、公正，或上级文件、学校制度明确禁止转专业的情形。”，其他的大一大二学生转出申请不设门槛。

但是否能转入，则有转入学院说的算，他们自己确定这么多申请转入的学生，我要哪些人。
