# 培养计划

## 什么是培养计划？
简单来说，培养计划就是不同的教学方案和教学方向

农大会给专业设置一至多种培养计划，以计算机为例，截止到本文写作时，计科专业的培养方案有三个方向：分别是软件方向、网安方向、智慧农业方向，看名字就能知道大致的选课内容

当然，选择某种培养计划，并不一定代表不能选择计划外的课程，比如网安方向的同学也可以选择Java相关的课程。（虽然笔者并不喜欢这门语言）

## 去哪找培养计划？
一般来说可以去各自学院的官网可以找到，比如机电院在“教学工作”->“本科生教育”->“培养方案”里，信科院在“人才培养”->“本科生教育”->“培养方案”里。

但有些学院在网上搜不到，可以去[教务系统](/云上农大/教务系统)里查看，点击“培养计划”就能看到自己这学期和之前学期的培养计划。
