# 买电脑
上大学了，大家应该都会得到自己人生中第一台电脑。

> 如果你很精通计算机相关知识，并且能够清晰的认识到自己需要什么，那么以下姑且可以不看。

## 什么时候买电脑？买什么电脑？
非常建议等知道自己是什么专业录取后再考虑买电脑。至于买什么电脑，可以多问一下自己本专业的学长学姐，让他们推荐（你永远不知道自己的专业课会用到什么奇奇怪怪的软件）。

如果你对电脑这一块什么都不懂，并且这将会是你人生中的第一台电脑，那可以买Windows电脑：不会出错的选择。

无论什么专业，都一定不要买华为的“纯血鸿蒙”电脑，无论你有多么支持华为（除非你已经有了别的电脑）。

尽量选择较大的品牌的电脑，而不是不小心买了一台山寨机。你可以让懂点的同学、学长、老师帮你挑，而不要闭上眼睛让家长随便买，无论是他们自己在网上搜还是找“熟人”。

## 上哪买电脑？
- 各品牌的京东自营
- 拼多多百亿补贴
- 天猫旗舰店
- 各品牌的线下直营门店（一般来说和网上同价）

不要随便找个电脑城就钻进去了，看你小白宰你没商量。
